Cos’è un torrent?

Un torrent è una traccia per il file e le sue fonti. Senza entrare troppo nei dettagli, è un file che ti permette di scaricare un contenuto digitale da più computer in una rete chiamata peer-to-peer, o come dice mia nonna, da pera a pera.

E’ facile trovare il file in questione su siti come Google. Il file in questione deve avere l’estensione .torrent e non essere chiaramente il titolo di una videocassetta per adulti, confrontabile dalla lista ufficiale. Se specificato inoltre ITA nei termini di ricerca, si può trovare il contenuto in italiano. Nota bene: non tutti i contenuti sono legalmente scaricabili in Italia (blink), ed esistono siti appositi per la ricerca dei file non protetti da copyright (blink blink).

Il file torrent non è il file completo! Il file .torrent rappresenta il film sulla rete e occupa veramente poco spazio, infatti deve essere aperto da un programma apposito che scaricherà i file da chiunque lo stia condividendo in quel momento, il che ci porta alla prossima sezione.

C’è una ultima considerazione da fare, che ci porta alla conclusione di questo articolo. Al contrario dei servizi di streaming, un download in peer-to-peer richiede che tutto il file sia completo prima della riproduzione. Richiede che tutte le parti del file siano state condivise e scaricate, visto che il download avviene a pacchetti in base a diversi fattori. Con i programmi tradizionali sopra elencati, i file sopra scaricati non potranno essere riprodotti fino al completamento, il che potrebbe indispettire il vostro partner, che non esiterà a insultarvi mentre trangugia pizza comodamente seduta sul divano, aspettando di lasciarvi.
