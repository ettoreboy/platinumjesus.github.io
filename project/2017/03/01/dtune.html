Dtune is my latest project, still in development. At the moment we have released a working beta that you can visit at dtune.online.

What does it do? It is a collaborative juke-box! Well, that does not really explain everything. The website is entirely static and hosted on github, it uses PubNub in order to publish song information from a streamers and coordinate voters. The desktop version acts as a manager, while the mobile version lists songs and allows the user to vote the next one. In the manager interface, you may login and import your favorite playlist form Youtube. If then you visit the same link on mobile devices or dtune mobile, you will be able to see your “stream”, and vote for the songs.
